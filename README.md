# Neeja - Passo Fundo

Os NEEJAs são Núcleos Estaduais de Educação de Jovens e Adultos, destinados àqueles
que não tiveram acesso ou continuidade de estudos no Ensino Fundamental e Ensino Médio na
idade própria (com oferta de cursos), com atendimento individualizado. Os NEEJAs funcionam em
escolas da Rede Pública Estadual em regime de aproveitamento de estudos.

## 🚀 QUEM PODE SE MATRICULAR?
  * Alunos com idades de 15 anos completos para o Ensino Fundamental e 18 anos completos para o Ensino Médio.
  * Alunos que não tenham concluído o Ensino Fundamental e Médio.
